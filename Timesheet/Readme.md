# Timesheet Focus ⏱️

A streamlined macOS/iOS utility built with **SwiftUI** and **SwiftData** to track work hours with native **iCloud** synchronization and **StoreKit 2** integration.

## 🌟 Features

- **SwiftData Persistence:** Modern, local-first data modeling for high-performance tracking.
- **iCloud Sync:** Automatic background synchronization across all your Apple devices via CloudKit.
- **StoreKit 2 Integration:** In-app purchase support for "Pro" features using the `products.storekit` configuration.
- **Native Performance:** 100% Swift and SwiftUI—no third-party dependencies.

## 🏗️ Technical Stack

- **Framework:** SwiftUI
- **Database:** SwiftData (SQLite backed)
- **Backend:** CloudKit (iCloud)
- **Testing:** StoreKit Configuration File for local subscription testing.
